# Contributing

## Setup

1. Install [Neovim](https://github.com/neovim/neovim/wiki/Installing-Neovim) and [Lazy.nvim](https://github.com/folke/lazy.nvim)
2. Clone this repo into `~/.local/share/nvim/site/pack/plugins/start/`
3. Run `:PackerSync`

## Committing

- Please ensure that you have a [ticket](https://github.com/users/TuringProblem/projects/10)
- when you are submitting a PR, please maje sure you have *ONE* approved reviewer
